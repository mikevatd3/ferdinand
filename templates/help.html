Ferdinand help

Ferdinand is a tool to manage definitions for analysis projects.

How to use this tool

Ferdinand has two concepts, sentences and phrases. A sentence is made up of phrases and each phrase is defined by a sentence. The tool provides simple but useful ways to manage and edit both.

Sentence analysis mode

Enter a statement or question from your analysis project into the box on the first page to begin. This takes you into 'sentence analysis' mode. As you review your sentence, there are likely phrases that are clear enough in plain language but will need further explaination or specificity as the analysis continues.

Find one of these phrases in the sentence. Highlight each word from that phrase by clicking it. Selected words will appear in orange. Once all the words from the phrase are selected, click the big blue botton to add the phrase to the phrase list. It can now be definied.

Defining phrases

Phrases can be defined on either the sentence analysis page in the phrase table at the bottom of the page, or on the phrases page that list all current phrases. Simply enter the phrase into the text block that appears when you click the edit link.

The definition that you provide for a phrase should be kept as simple as possible. The core trick to Ferdinand is that definitions are also sentences. This means that if there are phrases that are unclear in the next sentence down, you can add them to the list and define them with more sentences, and so on ...
